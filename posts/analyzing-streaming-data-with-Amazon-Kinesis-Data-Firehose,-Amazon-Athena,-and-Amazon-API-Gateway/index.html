Analyzing Streaming Data with Amazon Kinesis Data Firehose, Amazon Athena and Amazon API Gateway

Posted Sep 26, 2020 2020-09-26T08:00:00+03:00 by S. Aya

Introduction

Our objective is to setup a serverless system that will process, store and return stock market data. The main AWS services to be used for this task are Amazon Kinesis Firehose, Amazon KMS, Amazon Lambda, Amazon Athena and Amazon API Gateway.

We are going to use built-in sample data available on AWS as the data source. The data source returns data in the following format:

1

{"TICKER_SYMBOL":"QXZ","SECTOR":"HEALTHCARE","CHANGE":-0.05,"PRICE":84.51}

Load Data into Kinesis Firehose

Load the data into Kinesis Firehose; the way you do this is up to you. Then, your system should process the data in a way that it inserts information whether the tick is up or down trade (up trade is when CHANGE is positive, down trade is when CHANGE is negative) and also current timestamp in EST time zone. So the processed data should look like this:

1

{"TICKER_SYMBOL":"QXZ","SECTOR":"HEALTHCARE","CHANGE":-0.05,"PRICE":84.51, “TREND”:”DOWN”, “TIMESTAMP”:123456465132465}

Data Transformations with Lambda

In this part, our objective is to design the AWS architecture that will be capable of transforming the data that exists in Kinesis Firehose following the rules described earlier in this task. Transformed data is going to be encrypted using AWS KMS and then saved encrypted in Apache Parquet data format in S3 bucket. It is up to you which service you use to transform the data, as long as you will be writing an ETL job and saving the encrypted outcome in Apache Parquet data format.

import json
import calendar
import time
import base64

# Incoming Event
def lambda_handler(event, context):
    output = []
    now = calendar.timegm(time.gmtime())

    # Loop through records in incoming Event
    for record in event['records']:

        # Extract message
        message = json.loads(base64.b64decode(record['data']))

        if float(message['change']) < 0:
            trend = 'DOWN'
        elif float(message['change']) > 0:
            trend = 'POSITIVE'
        
        # Construct output
        data_field = {
            'timestamp': now,
            'trend' : trend,
            'sector': message['sector'],
            'change': float(message['change']),
            'ticker_symbol': message['ticker_symbol'],
            'price': float(message['price'])
        }
        
        output_record = {
            'recordId': record['recordId'],
            'result': 'Ok',
            'data': base64.b64encode(json.dumps(data_field))
        }
 
        output.append(output_record)

    return {'records': output}

Create an API endpoint using API Gateway

Finally, create an API endpoint using AWS Lambda and API Gateway to list the decrypted data stored in the S3 bucket. The API endpoint has to accept query parameters “from” and “to”, and it should query the data in this way. The endpoint can be implemented in your language of choice and ideally should make use of AWS Athena to query Parquet format. Roughly, the endpoint should look like this:

1

http://host/api/v1/ticks?from=[FROM_TIMESTAMP]&to=[TO_TIMESTAMP]

"use strict";

const AthenaExpress = require("athena-express"),
	AWS = require("aws-sdk");
	
const athenaExpressConfig = {
	aws: AWS,
	db: "default",
	getStats: true
};

const athenaExpress = new AthenaExpress(athenaExpressConfig);

exports.handler = async event => {
    let from = 1599326821
    let to = 1599326823
    
    if (event.queryStringParameters && event.queryStringParameters.from) {
        console.log("Received start timestamp: " + event.queryStringParameters.from);
        from = event.queryStringParameters.from;
    }
    
    if (event.queryStringParameters && event.queryStringParameters.to) {
        console.log("Received end timestamp: " + event.queryStringParameters.to);
        to = event.queryStringParameters.to;
    }
    
	const sqlQuery = "SELECT * FROM stock_table WHERE timestamp BETWEEN " + from + " AND " + to;

	try {
		let results = await athenaExpress.query(sqlQuery);
		return results;
		
	} catch (error) {
		return error;
	}
};
